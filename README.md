Compas
------
Downlaod mongoDB Compass:
wget https://downloads.mongodb.com/compass/mongodb-compass_1.25.0_amd64.deb
Install MongoDB Compass:
sudo dpkg -i mongodb-compass_1.25.0_amd64.deb
Start MongoDB Compass:
mongodb-compass
